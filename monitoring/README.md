# Monitoring

This directory contains the observability stack configuration for the project.

## Contents

- `docker-compose.yml`: Defines the services for Prometheus, Alertmanager, and Grafana.
- `prometheus.yml`: Prometheus configuration for scraping metrics.
- `alert_rules.yml`: Defines alerting rules for Prometheus.
- `alertmanager.yml`: Configuration for Alertmanager notifications.
- `grafana-datasources.yml`: Pre-configured data sources for Grafana.
- `grafana-dashboards/`: Directory containing Grafana dashboard definitions.
