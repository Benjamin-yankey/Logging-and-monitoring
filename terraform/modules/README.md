# Terraform Modules

This directory contains reusable Terraform modules used to build the project's infrastructure in a modular and maintainable way.

## Available Modules

- `vpc`: Networking infrastructure (VPC, Subnets, etc.).
- `security`: Security Groups and firewall rules.
- `ec2`: Application server provisioning.
- `jenkins`: Jenkins CI/CD server setup.
- `monitoring`: Dedicated monitoring infrastructure.
- `keypair`: SSH key management.
